Tecnologias utilizadas no projeto

- ChatGpt
- CapCut
- Heygen

Como foi feito ?

  - Primeiro gerei o roteiro no ChatGpt utilizando o histórico do Ebook Criado anteriormente
  - No Heygen gerei o vídeo com avatar
  - No Capcut fiz uma rápida edição
